three ways to run it

Pick services your way.

Interactive menu, direct service names, or just list what's available.

// interactive
Pick from a menu
Shows a numbered list of all services. Enter the numbers you want, or type 'all'.
sudo ./setup.sh --interactive
// direct
Name them directly
Know what you want? Pass the service names as arguments and it installs them immediately.
sudo ./setup.sh docker nginx postgres
// list
See what's available
Prints all available service names and their descriptions. No installation happens.
sudo ./setup.sh --list
